## Local Setup

    $ gem install jekyll # installs jekyll - http://jekyllrb.com/
    $ git clone https://github.com/ga-students/BEWD_DC_1
    $ cd BEWD_DC_1
    $ git checkout -b gh-pages origin/gh-pages
    $ jekyll serve --watch
    $ open http://localhost:4000/
